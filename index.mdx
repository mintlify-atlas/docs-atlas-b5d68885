# MS200 LiDAR Driver for ROS 2

A lightweight, native ROS 2 driver for the **MKS MS200 / Oradar MS200** Time-of-Flight LiDAR sensor. Written entirely in Python using `pyserial`, this driver implements the low-level serial communication protocol to decode sensor packets and publish standard `sensor_msgs/LaserScan` messages — with no proprietary SDKs required.

## Key features

<CardGroup cols={2}>
  <Card title="No proprietary SDK" icon="lock-open">
    Pure Python implementation using only `pyserial` — no closed-source binaries required
  </Card>
  <Card title="Standard ROS 2 output" icon="robot">
    Publishes `sensor_msgs/LaserScan` on `/scan`, compatible with Nav2 and SLAM Toolbox
  </Card>
  <Card title="Configurable parameters" icon="gear">
    Serial port, baudrate, and TF frame ID configurable via ROS 2 parameters at runtime
  </Card>
  <Card title="Automatic frame detection" icon="magnifying-glass">
    Detects the MS200 frame header `0x54 0x2C` to synchronize with the data stream
  </Card>
</CardGroup>
